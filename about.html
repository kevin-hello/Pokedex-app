About

Objective

To build a small web application with HTML, CSS, and JavaScript that loads data from an external API. And allows users to view details about selected data.

Key Features

  • -Load data from an external API
  • -View data as a list of items
  • -View details on an item with a click

Project Dependencies

  • -HTML
  • -CSS
  • -JavaScript
  • External API: https://pokeapi.co/
